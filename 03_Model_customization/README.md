# Model Customization

> **AWS event participants:** Model customization jobs (fine-tuning,
> continued pre-training, distillation) **cannot be run in
> AWS-managed event accounts** due to service quota restrictions.
> Skip the hands-on portion at AWS-run workshops and use it as
> a reference. To run these labs, use your own AWS account.

> **Note:** This folder is synced from
> [amazon-bedrock-samples/custom-models][src].
> Do not edit files here directly — changes will be overwritten.
> To contribute, open a PR in the source repository.
